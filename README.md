The goal of nCov2019\_analysis is to visualize the infection in Shanghai
and Guizhou province using Python and R.

I borrow from the data downloading from Lin (2020) and data manipulation
from Jian (2020).

## How to use

``` bash
make all
```

## Dependency

1.  Python 3.7.3
2.  R 3.6.0
3.  1.2.5033
